# ERP_Dev — 프로젝트 가이드

## 프로젝트 개요
소규모 사내용 ERP 시스템. 혼자 개발 및 유지보수.

## 기술 스택
- **언어**: Python
- **프레임워크**: Django
- **DB**: SQLite (개발) → PostgreSQL (배포 시 전환 예정)
- **UI**: Bootstrap 5

## 개발 환경
- 개발자 혼자 개발, Claude Code로 작업
- 여러 컴퓨터에서 작업 가능 (이 파일로 맥락 공유)
- GitHub: https://github.com/KimuSani/ERP_Dev

## UI/UX 요구사항
- **MDI 방식 (다중 창)**: 창을 여러 개 동시에 띄워서 나란히 볼 수 있어야 함 → `WinBox.js` 사용 (창 개수 제한 없음, 10개 이상도 가능)
- 기존 ERP 화면 레이아웃 최대한 유지 (직원 반발 최소화)
  - 왼쪽: 필터/검색 조건
  - 오른쪽: 결과 테이블
  - 상단 버튼: 목록보기 / 미리보기 / 인쇄 / 엑셀변환 / 종료
- 송장 출력: 브라우저 인쇄(Ctrl+P) 방식
